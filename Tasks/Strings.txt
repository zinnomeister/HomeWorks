Следующие задания буду связаны с работой со строками (String), так же необходимо научиться читать текст из файлов.
Для решения задач предпочтительно использовать официальную документацию (но не обязательно).

___________________

Задание #1:

Создать в коде string переменную, которая будет содержать текст в виде двух предложений.

В тексте должны быть:
  - лишние пробелы (например двойные или тройные)
  - запятые, которые не примыкают к словам
  - точки, которые не примыкают к словам
  - заглавные буквы там, где они быть не должны.
  - цифры
  - лишние восклицательные знаки

Задача состоит в том, чтобы нормализовать текст, чтобы бы он выглядел правильно отформатированным.

Пример входящего текста: 
"I`m doing my   homework Right now . Please , Don't    disturb me!!! 12345"

Программа должна убрать все недостатки из этого текста, и он должен выглядеть так:
"I`m doing my homework right now. Please, don't disturb me!"

Результат вывести в консоль.
___________________

Задание #2:

Создать в коде string переменную, которая будет содержать текст с пробелами.

Задача состоит в том, что необходимо распарсить (порезать) текст по пробелам,
и все части (слова) записать в массив string[].

После чего, вывести в консоль кол-во получившихся слов.

____________________

Задание #3

Создать текстовый файл в какой-то папке на компьютере.
Записать туда какой-то текст, чтобы были разные строки.

Задача состоит в том, чтобы прочитать данный текстовый файл, и вывести все прочитанные из файла строки в консоль.
